## Contributing

Contributions are welcome. Please open an issue to propose changes.

## Setting up the environment

This repository uses Gradle as its build tool.

To set up the repository, run:

```sh
$ ./gradlew build
```

This will install all the required dependencies and build the project.

## Modifying/Adding code

You can freely add or modify files.

## Running tests

```sh
$ ./scripts/test
```

## Linting and formatting

This repository uses standard linting and formatting tools.

To lint:

```sh
$ ./scripts/lint
```

To format:

```sh
$ ./scripts/format
```
